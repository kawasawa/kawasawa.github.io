## 背景

この VSCode のワークスペースには多くのソースコードやスクリプトがあり、その一部には "NOTE:" の文字列から始まるコメントが付いているものがあります。これらのコメントは技術的な観点で非常に有益です。

## 依頼

"NOTE:" の文字列から始まるコメントを RAG のように扱い、あなたのような生成 AI にとっての情報源としたいです。そのための簡易的な MCP サーバを構築してください。

## 補足

- キーワードである "NOTE:" については今後変更される可能性があります
- スクリプト等を実装する場合は Python 3 で記述してください
- この実装に対するユニットテストは不要です
- 極力サードパーティには頼らないでください
- 開発時の使用が目的であるため簡易的で構いません
- 勉強も兼ねているため分かりやすさを重視してください
